1. control
2. temp
3. vpd
4. co2
5. temp_vpd
6. vpd_co2
7. temp_co2
8. all

#### Gap-filling Method
Only used site-years with data gap sizes of 2-hr. 
Gap-filled those missing data points by linearly interpolating the value based on data from previous and post timepoints.
